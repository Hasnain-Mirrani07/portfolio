# Hasnain Mirrani — Portfolio

Senior Flutter Developer portfolio website built with React, TypeScript, Tailwind CSS, and Framer Motion.

## Live Demo

After deployment: `https://hasnain-mirrani07.github.io/portfolio/`

## Tech Stack

- **React 18** + **TypeScript**
- **Vite** — fast build tool
- **Tailwind CSS** — styling
- **Framer Motion** — animations
- **Lucide React** — icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

1. Push this repo to `https://github.com/Hasnain-Mirrani07/portfolio`
2. Go to **Settings → Pages → Build and deployment**
3. Set **Source** to **GitHub Actions**
4. Push to `main` branch — the workflow auto-deploys

## Deploy to Vercel / Netlify (Alternative)

1. Import the GitHub repo on [Vercel](https://vercel.com) or [Netlify](https://netlify.com)
2. Build command: `npm run build`
3. Output directory: `dist`
4. For Vercel/Netlify, change `base: '/'` in `vite.config.ts` (remove `/portfolio/`)
